## Problema

42 - Cria uma classe, com **um único** método estático `Main()`, que solicita
ao utilizador um número inteiro positivo e apresenta o respetivo número da
[sequência de Lucas](https://en.wikipedia.org/wiki/Lucas_number). O número
deve ser calculado de forma recursiva com uma ou mais **funções locais**.

## Soluções

### Solução 1

```csharp
using System;

namespace Ex_42
{
    class Program
    {
        static void Main(string[] args)
        {
            int inputN;
            int LucasSequence(int sequenceIndex)
            {
                if (sequenceIndex == 0) return 2;
                if (sequenceIndex == 1) return 1;

                return LucasSequence(sequenceIndex - 1)
                    + LucasSequence(sequenceIndex - 2);
            }

            Console.Write("Insert a positive Integer: ");
            for (; ; )
            {
                if (int.TryParse(Console.ReadLine(), out int result)
                    && result > 0)
                {
                    inputN = result;
                    break;
                }
                else
                    Console.Write("Insert a positive Integer: ");
            }



            Console.WriteLine(LucasSequence(inputN));
        }
    }
}
```
