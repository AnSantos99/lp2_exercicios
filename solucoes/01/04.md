## Problema

3 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua
resposta.

```cs
byte b = byte.MaxValue - 1;

for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Iteração {i}: {(byte) (++b - 1)}");
}
```

## Soluções

### Solução 1

Ecrã:
  8
  6
104

Isto porque:
```cs
if (v % 2 == 0)
```
So entra no if quando o ```v``` é multiplo de 2 e
```cs
Console.WriteLine("{0,3}", v);
```
cria um espaçamento à esquerda.
